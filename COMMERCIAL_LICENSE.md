# Licencia Comercial

Este proyecto se distribuye bajo `PolyForm Noncommercial 1.0.0` para uso no comercial.

Si deseas usar este software con fines comerciales (venta, SaaS, servicios pagados, integracion comercial o cualquier actividad con lucro), debes obtener permiso previo por escrito del titular.

## Como solicitar permiso comercial

Canal de contacto oficial: este mismo repositorio en GitHub.

1. Abre un **Issue** o **Discussion** en este repo.
2. Usa el titulo: `Commercial License Request`.
3. Incluye:
   - nombre de persona o empresa,
   - caso de uso comercial,
   - forma de distribucion,
   - pais o region de operacion,
   - estimacion de usuarios o clientes.

Sin autorizacion expresa por escrito, el uso comercial no esta permitido.
